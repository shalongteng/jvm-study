#10、JDK 8
    2014年3月18日
    JDK 8提供了那些曾在JDK 7中规划过，但最终未能在JDK 7中完成的功能，主要包括：
    ·JEP 126：对Lambda表达式的支持，这让Java语言拥有了流畅的函数式表达能力。
    ·JEP 104：内置Nashorn JavaScript引擎的支持。
    ·JEP 150：新的时间、日期API。
    ·JEP 122：彻底移除HotSpot的永久代。
